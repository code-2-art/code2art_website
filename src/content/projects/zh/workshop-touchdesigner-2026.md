## 活动概述

TouchDesigner 是实时视觉创作领域最主流的工具之一，广泛应用于新媒体演出、沉浸式展览和交互装置。本次工作坊将在两天内带领学员从零搭建一套完整的实时视觉系统。

## 时间与地点

- **日期**：2026 年 3 月 15 — 16 日（周六、周日）
- **时间**：10:00 — 17:00（含午休）
- **地点**：线上 Zoom + 录播回放

## 课程大纲

### Day 1 — 基础与逻辑

| 时间 | 内容 |
|------|------|
| 10:00 | TouchDesigner 界面与节点系统 |
| 11:00 | TOP / CHOP / SOP 三大家族 |
| 14:00 | Noise、Feedback 与生成纹理 |
| 16:00 | 练习：制作一组动态视觉 Loop |

### Day 2 — 进阶与演出

| 时间 | 内容 |
|------|------|
| 10:00 | 音频响应：用声音驱动画面 |
| 11:00 | GLSL Shader 在 TD 中的用法 |
| 14:00 | 多屏输出与 NDI 推流 |
| 16:00 | 综合练习：搭建一套 Live Set |

## 适合谁参加

- 对实时视觉 / VJ 感兴趣的创作者
- 希望将编程技能延伸到视觉表演领域的开发者
- 新媒体艺术专业学生
